## 上手指南
以下指南将帮助你在本地机器上安装和运行该项目，进行开发和测试。
#### 安装要求
运行该项目必须要具备的条件及必须要安装的软件。
- 任意浏览器（需支持JavaScript、CSS，一般2020年以后的浏览器都支持）
- 下载后，**无需**互联网，即可运行
